These are some of the things I've done.

My primary development experience revolves around Wordpress, the technologies it uses, and the server environments it lives on. Almost every site that I build is using the Wordpress platform and a custom theme I develop from the ground up.

I will write custom functionality over installing a plugin whenever I can. There's nothing wrong with some plugins, I've even written a few custom plugins, but when a simple function can replace a plugin I do it every time. I tend to look for effeciencies like this with every aspect of my development process. The web moves fast, sometimes so fast that every new project I pick up is configured completely different. Modernizr and CSS compilers have changed our stylesheets forever and I jumped on board as quick as I could. GIT versioning is one of the best tools available and will continue to provide peace of mind for coders and site owners alike.

Web Host & DNS management are a few more important skills I've accrued over time. I've learned that there are no shortcuts here and using a reputable Web Host and a properly configured DNS is a must. I've worked with many web hosts and registrars now and can vouch first hand that 'you get what you pay for'. Most of the sites I've worked on have been on LAMP (Linux, Apache, MySQL, PHP) stacks in various forms. I've setup dedicated servers from scratch, used VPS solutions, and even have had a few sites on shared hosting. Recently, the switch from Apache to NGINX and PHP 7.x have given our Wordpress sites some serious performance boosts. This combined with proper image optimization, caching and code minification really sets them up for a good rating when Google checks the page speed.

Over the past 13 years I've worked pretty closely with my designer, project manager and sales staff and have picked up skills from them whenever possible. I've learned a lot about purposeful design and effective UI implementation to keep site user experiences as painless as possible. I regularly take the feedback from the experiences with my colleagues and clients and adjust my development process accordingly. I like to think I'm good at what I do and I will never get comfortable with that thought because I know the moment I stop improving my craft, I'll already be behind the competition.
